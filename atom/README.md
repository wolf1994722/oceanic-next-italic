# Oceanic Next _Italic_ for Atom

## Installation

### Atom Package Manager (APM)
```bash
apm install oceanic-next-italic-syntax
```

### Git clone
```bash
cd ~/.atom/packages
git clone https://github.com/SintrumIT/oceanic-next-italic --depth=1
```
